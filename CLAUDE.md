# CLAUDE.md

## Design System

Always read `DESIGN.md` before making visual or UI decisions.

All font choices, colors, spacing, layout rules, motion, accessibility, and product-specific UI constraints are defined there.

Do not deviate from `DESIGN.md` without explicit user approval.

When reviewing or QA testing UI, flag any code that does not match `DESIGN.md`.

## Privacy Boundary

The public AlloIQ site may show portfolio weights and portfolio-weight research deltas.

It must not show share counts, broker account names, dollar values, transaction details, or private raw position rows.
